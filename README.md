# AI-Powered Talent Intelligence & Workforce Optimization Suite (T-IQ)

## Project Overview
T-IQ is a unified AI system designed to help HR teams optimize workforce management, improve hiring efficiency, predict attrition, and enhance employee engagement.

This project demonstrates:
- Data cleaning, EDA, and preprocessing of HR datasets
- ML, NLP, and deep learning pipelines for HR tasks
- Resume parsing, skill matching, sentiment analysis, and chatbot integration
- Dashboarding and reporting for HR decision-making

---

## Skills You Will Learn
- Python fundamentals: loops, functions, classes, file handling
- Data handling with Pandas, JSON, and complex nested files
- SQL operations: joins, windows, ACID transactions
- EDA with plots, distributions, and statistical summaries
- Machine Learning: classification, regression, clustering, recommendation
- Dimensionality reduction: PCA, LDA
- Deep learning with PyTorch (MLP, CNN, RNN, LSTM)
- NLP: Transformers for resume parsing, sentiment analysis, and summarization
- LLM integration for HR chatbots and summarization
- Prompt engineering
- End-to-end deployment of HR AI solutions

## Datasets
- IBM HR Analytics Attrition dataset  
- Resume dataset (CSV + PDFs)  
- Job descriptions dataset  
- Glassdoor/Indeed reviews dataset  
- HRMS synthetic dataset (JSON)  
- Interview transcripts (synthetic, JSON)

> **Note:** Raw datasets are **not included** in the GitHub repository due to size and privacy.
